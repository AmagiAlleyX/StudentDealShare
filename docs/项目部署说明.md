## 2. 数据库部署

### 2.1 MySQL 安装

#### 2.1.1 CentOS 安装
```bash
# 下载 MySQL YUM 源
wget https://dev.mysql.com/get/mysql80-community-release-el7-3.noarch.rpm

# 安装 YUM 源
sudo rpm -Uvh mysql80-community-release-el7-3.noarch.rpm

# 安装 MySQL 服务器
sudo yum install -y mysql-server

# 启动 MySQL
sudo systemctl start mysqld
sudo systemctl enable mysqld

# 获取临时密码
sudo grep 'temporary password' /var/log/mysqld.log
```

#### 2.1.2 Ubuntu 安装
```bash
# 下载 MySQL APT 源
wget https://dev.mysql.com/get/mysql-apt-config_0.8.17-1_all.deb

# 安装 APT 源
sudo dpkg -i mysql-apt-config_0.8.17-1_all.deb

# 更新包索引
sudo apt update

# 安装 MySQL 服务器
sudo apt install -y mysql-server

# 启动 MySQL
sudo systemctl start mysql
sudo systemctl enable mysql
```

### 2.2 MySQL 配置

#### 2.2.1 修改配置文件
```bash
sudo vim /etc/my.cnf
```

添加以下配置：
```ini
[mysqld]
# 基本配置
server_id=1
port=3306
basedir=/usr/local/mysql
datadir=/var/lib/mysql
socket=/var/lib/mysql/mysql.sock

# 字符集
character-set-server=utf8mb4
collation-server=utf8mb4_general_ci

# 连接数
max_connections=500
max_connect_errors=100

# 性能优化
innodb_buffer_pool_size=2G
innodb_log_file_size=256M
innodb_flush_log_at_trx_commit=2
innodb_flush_method=O_DIRECT

# 日志
log-error=/var/log/mysql/error.log
slow_query_log=1
slow_query_log_file=/var/log/mysql/slow.log
long_query_time=2

# 主从复制（从库配置）
# server_id=2
# relay_log=/var/log/mysql/relay-bin.log
# read_only=1
```

#### 2.2.2 重启 MySQL
```bash
sudo systemctl restart mysqld
```

### 2.3 创建数据库和用户

```sql
-- 登录 MySQL
mysql -u root -p

-- 创建数据库
CREATE DATABASE IF NOT EXISTS student_deal_share 
DEFAULT CHARACTER SET utf8mb4 
DEFAULT COLLATE utf8mb4_general_ci;

-- 创建用户
CREATE USER 'dealshare'@'%' IDENTIFIED BY 'YourPassword123!';

-- 授权
GRANT ALL PRIVILEGES ON student_deal_share.* TO 'dealshare'@'%';

-- 刷新权限
FLUSH PRIVILEGES;

-- 退出
EXIT;
```

### 2.4 导入数据库结构

```bash
# 执行 SQL 脚本
mysql -u dealshare -p student_deal_share < /path/to/schema.sql
```

### 4.1 构建项目


### 4.2 配置文件

创建生产环境配置文件 `application-prod.yml`：

```yaml
server:
  port: 8080

spring:
  datasource:
    url: jdbc:mysql://localhost:3306/student_deal_share?useUnicode=true&characterEncoding=utf8&useSSL=false&serverTimezone=Asia/Shanghai
    username: dealshare
    password: YourPassword123!
    driver-class-name: com.mysql.cj.jdbc.Driver
    
  redis:
    host: localhost
    port: 6379
    password: YourRedisPassword123!
    database: 0
    timeout: 3000ms
    lettuce:
      pool:
        max-active: 20
        max-idle: 10
        min-idle: 5

  servlet:
    multipart:
      max-file-size: 10MB
      max-request-size: 10MB

# JWT 配置
jwt:
  secret: YourJWTSecretKey2024!
  expiration: 7200000  # 2 小时
  refresh-expiration: 604800000  # 7 天

# 文件上传配置

  upload-path: /data/uploads
  max-size: 10485760  # 10MB

# 日志配置

  level:
    root: INFO
    com.student.dealshare: INFO
  file:
    name: /var/log/student-deal-share/app.log
  logback:
    rollingpolicy:
      max-file-size: 100MB
      max-history: 30
```

### 4.3 运行应用

#### 4.3.1 直接运行
```bash
java -jar target/student-deal-share-1.0.0.jar --spring.profiles.active=prod
```

#### 4.3.2 使用脚本运行
创建启动脚本 `start.sh`：
```bash
#!/bin/bash

APP_NAME="student-deal-share"
APP_JAR="student-deal-share-1.0.0.jar"
APP_PROFILE="prod"
LOG_PATH="/var/log/student-deal-share"
PID_FILE="$APP_NAME.pid"

# 检查是否已运行
if [ -f "$PID_FILE" ]; then
    PID=$(cat $PID_FILE)
    if ps -p $PID > /dev/null; then
        echo "$APP_NAME is already running (PID: $PID)"
        exit 1
    fi
fi

# 创建日志目录
mkdir -p $LOG_PATH

# 启动应用
nohup java -jar \
    -Xms512m \
    -Xmx1024m \
    -XX:+UseG1GC \
    $APP_JAR \
    --spring.profiles.active=$APP_PROFILE \
    > $LOG_PATH/startup.log 2>&1 &

# 保存 PID
echo $! > $PID_FILE

echo "$APP_NAME started successfully (PID: $!)"
```

赋予执行权限：
```bash
chmod +x start.sh
```

#### 4.3.3 配置 systemd 服务
创建服务文件：
```bash
sudo vim /etc/systemd/system/student-deal-share.service
```

内容：
```ini
[Unit]
Description=Student Deal Share Application
After=syslog.target network.target

[Service]
User=root
Group=root
WorkingDirectory=/opt/student-deal-share
ExecStart=/usr/bin/java -jar -Xms512m -Xmx1024m student-deal-share-1.0.0.jar --spring.profiles.active=prod
SuccessExitStatus=143
Restart=always
RestartSec=10
StandardOutput=journal
StandardError=journal
SyslogIdentifier=student-deal-share

[Install]
WantedBy=multi-user.target
```

启动服务：
```bash
# 重载 systemd
sudo systemctl daemon-reload

# 启动服务
sudo systemctl start student-deal-share

# 设置开机自启
sudo systemctl enable student-deal-share

# 查看状态
sudo systemctl status student-deal-share

# 查看日志
sudo journalctl -u student-deal-share -f
```

#### 5.1.2 Nginx 配置
```bash
sudo vim /etc/nginx/conf.d/student-deal-share.conf
```

配置内容：
```nginx
server {
    listen 80;
    server_name admin.dealshare.com;
    
    root /opt/student-deal-share/web;
    index index.html;
    
    # Gzip 压缩
    gzip on;
    gzip_min_length 1k;
    gzip_comp_level 6;
    gzip_types text/plain application/javascript application/x-javascript text/css application/xml text/javascript application/json;
    
    # 静态资源缓存
    location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg|woff|woff2|ttf|eot)$ {
        expires 30d;
        add_header Cache-Control "public, immutable";
    }
    
    # SPA 路由支持
    location / {
        try_files $uri $uri/ /index.html;
    }
    
    # API 代理
    location /api/ {
        proxy_pass http://localhost:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }
    
    # 日志
    access_log /var/log/nginx/student-deal-share-access.log;
    error_log /var/log/nginx/student-deal-share-error.log;
}
```

## 6. Docker 部署（可选）

### 6.1 创建 Dockerfile

```dockerfile
FROM openjdk:17-slim

WORKDIR /app

COPY target/student-deal-share-1.0.0.jar app.jar

EXPOSE 8080

ENV JAVA_OPTS="-Xms512m -Xmx1024m -XX:+UseG1GC"

ENTRYPOINT java $JAVA_OPTS -jar app.jar --spring.profiles.active=prod
```

### 6.2 创建 docker-compose.yml

```yaml
version: '3.8'

services:
  mysql:
    image: mysql:8.0
    container_name: dealshare-mysql
    environment:
      MYSQL_ROOT_PASSWORD: rootPassword123!
      MYSQL_DATABASE: student_deal_share
      MYSQL_USER: dealshare
      MYSQL_PASSWORD: dealsharePassword123!
    volumes:
      - mysql-data:/var/lib/mysql
      - ./sql:/docker-entrypoint-initdb.d
    ports:
      - "3306:3306"
    networks:
      - dealshare-network
    restart: always

  redis:
    image: redis:7-alpine
    container_name: dealshare-redis
    command: redis-server --requirepass redisPassword123!
    volumes:
      - redis-data:/data
    ports:
      - "6379:6379"
    networks:
      - dealshare-network
    restart: always

  backend:
    build: ../backend
    container_name: dealshare-backend
    depends_on:
      - mysql
      - redis
    ports:
      - "8080:8080"
    environment:
      SPRING_PROFILES_ACTIVE: prod
      DB_HOST: mysql
      DB_PORT: 3306
      REDIS_HOST: redis
      REDIS_PORT: 6379
    volumes:
      - uploads:/data/uploads
      - logs:/var/log/student-deal-share
    networks:
      - dealshare-network
    restart: always

  nginx:
    image: nginx:alpine
    container_name: dealshare-nginx
    depends_on:
      - backend
    ports:
      - "80:80"
    volumes:
      - ./nginx.conf:/etc/nginx/conf.d/default.conf
      - ../frontend/web/dist:/usr/share/nginx/html
    networks:
      - dealshare-network
    restart: always

volumes:
  mysql-data:
  redis-data:
  uploads:
  logs:

networks:
  dealshare-network:
    driver: bridge
```

### 6.3 启动容器

## 8. 备份与恢复

### 8.1 数据库备份

#### 8.1.1 手动备份
```bash
# 备份整个数据库
mysqldump -u dealshare -p student_deal_share > backup_$(date +%Y%m%d).sql

# 压缩备份
mysqldump -u dealshare -p student_deal_share | gzip > backup_$(date +%Y%m%d).sql.gz
```

#### 8.1.2 定时备份
创建备份脚本 `backup.sh`：
```bash
#!/bin/bash

BACKUP_DIR="/data/backup/mysql"
DATE=$(date +%Y%m%d_%H%M%S)
DB_NAME="student_deal_share"
DB_USER="dealshare"
DB_PASS="YourPassword123!"

mkdir -p $BACKUP_DIR

mysqldump -u$DB_USER -p$DB_PASS $DB_NAME | gzip > $BACKUP_DIR/backup_$DATE.sql.gz

# 删除 30 天前的备份
find $BACKUP_DIR -name "backup_*.sql.gz" -mtime +30 -delete

echo "Backup completed: backup_$DATE.sql.gz"
```

设置定时任务：
```bash
crontab -e

# 每天凌晨 2 点备份
0 2 * * * /path/to/backup.sh
```

### 8.2 数据库恢复

```bash
# 解压并恢复
gunzip < backup_20240101.sql.gz | mysql -u dealshare -p student_deal_share
```

## 9. 常见问题

### 9.1 启动失败

**问题**: 应用启动失败
**排查步骤**:
1. 查看日志：`journalctl -u student-deal-share -f`
2. 检查端口占用：`netstat -tlnp | grep 8080`
3. 检查数据库连接：`telnet localhost 3306`
4. 检查 Redis 连接：`telnet localhost 6379`

### 9.2 数据库连接失败

**问题**: 无法连接数据库
**排查步骤**:
1. 检查 MySQL 服务状态：`systemctl status mysqld`
2. 检查用户权限：`SHOW GRANTS FOR 'dealshare'@'%';`
3. 检查防火墙：`firewall-cmd --list-all`

### 9.3 内存溢出

**问题**: OutOfMemoryError
**解决方案**:
1. 调整 JVM 参数：`-Xms512m -Xmx1024m`
2. 检查代码内存泄漏
3. 增加服务器内存
