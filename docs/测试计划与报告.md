# 大学生资源信息分享平台 - 测试计划与测试报告

## 第一部分：测试计划

### 1. 测试目标

确保大学生资源信息分享平台的功能完整性、稳定性、安全性和性能，具体目标包括：
- 验证所有功能需求是否实现
- 确保系统在各种场景下稳定运行
- 发现并修复潜在缺陷
- 验证系统性能是否满足要求
- 确保系统安全性

### 2. 测试范围

- SQL 注入测试

**测试框架**: JUnit 5 + Mockito

**测试范围**:
- Service 层业务逻辑
- Mapper 层数据库操作

**覆盖率要求**:
- 核心业务代码覆盖率 ≥ 80%
- 总体代码覆盖率 ≥ 70%

**测试用例示例**:

```java
@SpringBootTest
class UserServiceTest {

    @Autowired
    private UserService userService;

    @MockBean
    private UserMapper userMapper;

    @Test
    @DisplayName("测试用户注册 - 成功")
    void testRegisterSuccess() {
        // 准备测试数据
        RegisterDTO registerDTO = new RegisterDTO();
        registerDTO.setUsername("testuser");
        registerDTO.setPassword("123456");
        registerDTO.setPhone("13800138000");
        registerDTO.setCode("123456");

        // Mock 数据
        when(userMapper.selectByUsername("testuser")).thenReturn(null);
        when(userMapper.selectByPhone("13800138000")).thenReturn(null);

        // 执行测试
        UserVO result = userService.register(registerDTO);

        // 验证结果
        assertNotNull(result);
        assertEquals("testuser", result.getUsername());
    }

    @Test
    @DisplayName("测试用户注册 - 用户名已存在")
    void testRegisterUsernameExists() {
        // 准备测试数据
        RegisterDTO registerDTO = new RegisterDTO();
        registerDTO.setUsername("testuser");
        registerDTO.setPassword("123456");
        registerDTO.setPhone("13800138000");

        // Mock 数据
        when(userMapper.selectByUsername("testuser")).thenReturn(new User());

        // 执行测试并验证异常
        assertThrows(BusinessException.class, () -> {
            userService.register(registerDTO);
        });
    }
}
```

**Web 端测试用例** (Selenium):

```java
@SpringBootTest
public class LoginTest {

    private WebDriver driver;

    @BeforeEach
    public void setUp() {
        System.setProperty("webdriver.chrome.driver", "path/to/chromedriver");
        driver = new ChromeDriver();
        driver.manage().window().maximize();
        driver.get("http://localhost:8080/login");
    }

    @Test
    @DisplayName("测试登录 - 成功")
    public void testLoginSuccess() {
        // 输入用户名
        WebElement usernameInput = driver.findElement(By.id("username"));
        usernameInput.sendKeys("testuser");

        // 输入密码
        WebElement passwordInput = driver.findElement(By.id("password"));
        passwordInput.sendKeys("123456");

        // 点击登录按钮
        WebElement loginButton = driver.findElement(By.id("loginBtn"));
        loginButton.click();

        // 验证跳转到首页
        WebDriverWait wait = new WebDriverWait(driver, Duration.ofSeconds(10));
        WebElement welcomeElement = wait.until(ExpectedConditions
            .visibilityOfElementLocated(By.id("welcome")));
        assertEquals("欢迎回来", welcomeElement.getText());
    }

    @Test
    @DisplayName("测试登录 - 密码错误")
    public void testLoginWrongPassword() {
        // 输入用户名
        WebElement usernameInput = driver.findElement(By.id("username"));
        usernameInput.sendKeys("testuser");

        // 输入错误密码
        WebElement passwordInput = driver.findElement(By.id("password"));
        passwordInput.sendKeys("wrongpassword");

        // 点击登录按钮
        WebElement loginButton = driver.findElement(By.id("loginBtn"));
        loginButton.click();

        // 验证错误提示
        WebDriverWait wait = new WebDriverWait(driver, Duration.ofSeconds(10));
        WebElement errorElement = wait.until(ExpectedConditions
            .visibilityOfElementLocated(By.className("error-message")));
        assertTrue(errorElement.getText().contains("用户名或密码错误"));
    }

    @AfterEach
    public void tearDown() {
        if (driver != null) {
            driver.quit();
        }
    }
}
```

**移动端测试用例** (Appium):

```java
@SpringBootTest
public class DealPublishTest {

    private AppiumDriver driver;

    @BeforeEach
    public void setUp() {
        // 配置 Appium
        DesiredCapabilities caps = new DesiredCapabilities();
        caps.setCapability("platformName", "HarmonyOS");
        caps.setCapability("appPackage", "com.student.dealshare");
        caps.setCapability("appActivity", ".MainActivity");
        
        driver = new AndroidDriver(new URL("http://localhost:4723/wd/hub"), caps);
    }

    @Test
    @DisplayName("测试发布优惠流程")
    public void testPublishDeal() {
        // 点击发布按钮
        WebElement publishBtn = driver.findElement(By.id("publish_btn"));
        publishBtn.click();

        // 输入标题
        WebElement titleInput = driver.findElement(By.id("deal_title"));
        titleInput.sendKeys("京东 618 大促");

        // 输入描述
        WebElement descInput = driver.findElement(By.id("deal_description"));
        descInput.sendKeys("满 199 减 100");

        // 选择分类
        WebElement categorySpinner = driver.findElement(By.id("category_spinner"));
        categorySpinner.click();
        WebElement categoryOption = driver.findElement(By.xpath("//android.widget.TextView[@text='电商优惠']"));
        categoryOption.click();

        // 点击发布
        WebElement submitBtn = driver.findElement(By.id("submit_btn"));
        submitBtn.click();

        // 验证发布成功提示
        WebDriverWait wait = new WebDriverWait(driver, Duration.ofSeconds(10));
        WebElement toast = wait.until(ExpectedConditions
            .visibilityOfElementLocated(By.className("android.widget.Toast")));
        assertTrue(toast.getText().contains("发布成功"));
    }

    @AfterEach
    public void tearDown() {
        if (driver != null) {
            driver.quit();
        }
    }
}
```
