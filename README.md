# 🎓 CampusEcoHub — 大学生资源信息分享平台（经济）

> 让每一次优惠都不被错过，让每一份经验都能被看见。

## 📌 项目背景

当前电商平台、品牌方的营销活动形式多样（预售抢购、优惠券薅羊毛、限时折扣等），但普通用户获取优质活动信息的渠道分散，且抢购环节易因操作慢、信息差错失福利；同时，很多用户有分享自己“薅羊毛”经验、抢购成功案例的需求，却缺乏专属的垂直分享平台。

**CampusEcoHub** 专为大学生群体打造，聚合校园可触达的优惠信息，并提供经验分享与抢购互助能力。

---

## ✨ 核心功能

### 🔍 优惠信息聚合
- 活动分类（数码 / 生活 / 外卖 / 出行 / 课程）
- 来源标注（京东 / 淘宝 / 美团 / 饿了么 / 银行APP等）
- 时效性提醒（倒计时 / 即将开始）

### 💬 薅羊毛经验分享
- 用户发帖（图文 / 链接）
- 抢购成功案例与技巧
- 评论互动与避坑提醒

### ⏰ 抢购协作
- 活动收藏与提醒
- 抢购前预检清单
- （可选）互助抢购请求

### 🧑‍🎓 校园特色
- 校内周边优惠（打印 / 食堂 / 超市）
- 学生认证专属活动推荐
- 按学校/地区筛选
